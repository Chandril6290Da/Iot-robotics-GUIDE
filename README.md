# 🚀 IoT & Robotics — Complete Learning Resource Map

An interactive web-based guide for learning **IoT (Internet of Things)** and **Robotics** — from beginner to industry level.

🔗 This project combines:

* 📺 YouTube learning channels (Hindi & English)
* 💻 GitHub repositories
* 📊 Structured learning roadmap (Beginner → Industry)

---

## 🌐 Live Demo

👉 https://chandril6290da.github.io/Iot-robotics-GUIDE/

---

## 📌 Features

✨ Clean modern UI (dark theme)
🔍 Smart search (channels, repos, topics)
🎯 Filters:

* Content Type (YouTube / GitHub)
* Language (Hindi / English / Both)
* Skill Level (Beginner → Industry)

📚 Covers 10 major domains:

* Electronics & Hardware
* Embedded Systems
* Sensors & Actuators
* IoT Networking
* Cloud & Data
* Robotics
* AI / ML / Computer Vision
* Industrial Automation
* IoT Cybersecurity
* Autonomous Systems & Drones

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Custom UI + Responsive Design)
* Vanilla JavaScript (No frameworks)

## 🚀 Deployment (GitHub Pages)

This project is deployed using **GitHub Pages + GitHub Actions**.

### Steps:

1. Push code to `main` branch
2. Ensure file name is:

   ```
   index.html
   ```
3. GitHub Actions will auto-deploy

## 📖 How to Use

1. Open the website
2. Select filters (Language / Level / Type)
3. Search for topics like:

   * "ESP32"
   * "ROS2"
   * "YOLO"
4. Click cards to open:

   * YouTube channel
   * GitHub repository
